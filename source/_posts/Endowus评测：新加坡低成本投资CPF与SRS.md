在新加坡想投资，但不想天天盯盘、也不想被高额销售佣金「割」？**Endowus** 是本地少数能同时打理 **现金、CPF 普通户（OA）、SRS** 的数字财富平台，受新加坡金管局（MAS）监管。综合 [SingSaver](https://www.singsaver.com.sg/investment/blog/endowus-review-singapore)、[GrowBeansprout](https://growbeansprout.com/endowus-review) 等近期评测，它最适合「长期持有、省心定投」的人——而不是短线炒股的选手。

| 你的情况 | 是否适合 Endowus |

## Endowus 是什么？

Endowus 成立于 2017 年，是新加坡持牌的 **fee-only（仅收咨询费）** 数字财富平台，不赚基金销售佣金，把基金管理人支付的 trailer fee **100% 返现**给客户。平台接入 PIMCO、Dimensional、Schroders 等机构级基金，提供 250+ 只互惠基金（Unit Trust），资产托管在 UOB Kay Hian 以客户名义的独立信托账户。

![合作基金管理人](https://cdn.growbeansprout.com/strapi-uploads/Endowus_Partners_21_Jan_2025_b39126c31d.jpg)

和传统券商 DIY 买股票不同，Endowus 走的是 **「帮你配组合 + 自动再平衡」** 路线，更像智能投顾，但基金选择比多数竞品更宽。

## 核心优势（评测共识）

### 1. 现金、CPF、SRS 一个账户搞定

| 资金来源 | 典型用途 | 平台费率（Endowus Access Fee） |
| --- | --- | --- |
| 现金 | 长期财富积累、Income 收息组合 | 按资产规模阶梯，**0.25%–0.60%/年** |
| CPF OA / SA | 让退休储蓄跑赢 OA 2.5% 基准利率 | 多基金组合 **0.40%/年**；单基金 **0.30%/年** |
| SRS | 省税 + 长期投资 | 同 CPF，固定费率、无阶梯 |
| 短期闲钱 | Cash Smart 现金管理 | 统一 **0.15%/年** |

> CPF/SRS 费率不与现金资产合并计算阶梯，但费率本身已经偏低。CPF OA 目前仍是本地智能投顾里较突出的选项——不必自己跑银行 CPF 投资户 + 券商两套流程。

### 2. 费率透明，没有隐藏销售佣金

| 费用类型 | Endowus 怎么收 |
| --- | --- |
| 开户 / 账户维护 | **免费** |
| 申购 / 赎回 / 转换 | **$0** 销售佣金、$0 交易费 |
| 平台服务费（Access Fee） | 0.15%–0.60%/年，已含托管、再平衡、咨询 |
| Trailer Fee 回扣 | **100% 返现**，降低实际基金层面成本 |
| 基金本身管理费 | 已反映在净值中，通常 0.05%–1%/年 |

**全产品费率一览**（数据整理自 [Endowus 官方定价页](https://endowus.com/pricing)，另加 9% GST）：

| 组合类型 | 现金（Cash） | SRS | CPF OA / SA |

> CPF / SRS 与现金资产**分开计费**，不合并计算阶梯。通过 CPF 投资时，代理银行（DBS / UOB / OCBC）每笔交易另收约 **$2–$2.50** 手续费。

$10,000 现金、多基金组合：年费约 $60 + GST ≈ **$65**，月均 $5 出头。相比传统理财顾问 1.5%–2% 的综合成本，长期复利差距可观。

### 4. 用起来省心

- **SingPass / MyInfo** 几分钟开户
- 支持 **eGIRO** 自动定投（周 / 双周 / 月）
- **无锁定期**，随时赎回（到账通常 2–5 个工作日）
- 定期再平衡，减少「追涨杀跌」的冲动操作
- 提供 CPF 退休测算工具，帮你看懂 OA 投出去后的长期差异

## 需要注意的几点（客观短板）

| 问题 | 说明 |
| --- | --- |
| **最低 $1,000 起投** | 比 Syfe、StashAway 等门槛高，后续追加每次至少 $100 |
| **小额现金费率偏高** | 20 万以下现金费率 0.60%，小资金要算清楚是否划算 |
| **赎回比股票慢** | 卖出后 2–5 个工作日到账，应急金应用 Cash Smart 而非长期组合 |
| **投资有风险** | CPF OA 默认 2.5% 是无风险基准利率；市场组合可能出现波动和回撤 |
| **不服务美国税务居民** | 美国公民 / 绿卡持有者目前无法开户 |

社区里也有资深用户讨论：如果只买单一低成本指数基金，直接走 POEMS 等券商可能更便宜；但 **Endowus 的价值在于「CPF/SRS 一站式 + 机构基金准入 + 自动再平衡」**，省的是时间和精力，不只是费率。

## 和同类平台怎么选？

| 维度 | Endowus | 典型竞品（StashAway / Syfe） |
| --- | --- | --- |
| CPF OA 投资 | ✅ 支持 | ❌ 一般不支持 |
| 底层资产 | 互惠基金为主 | 多为 ETF |
| 自选基金 | ✅ Fund Smart | 有限 |
| 现金起投门槛 | $1,000 | 部分平台更低或为零 |
| 费率区间 | 0.15%–0.60% | 约 0.2%–0.65% |

如果你核心诉求是 **「CPF 钱生钱」**，Endowus 在本地仍属优先选项；若只是小额现金试水，可以对比其他低门槛平台。

## 开户流程（四步）

1. 用 SingPass 在 [Endowus 官网](https://endowus.com) 或 App 注册

4. 通过 eGIRO 或转账入金 ≥ $1,000，确认买入

## 适合谁总结

**值得考虑**：有稳定收入、能持有 3 年以上、想让 CPF OA / SRS 跑赢默认利息、又不想自己盯盘的在新加坡工作者（含 PR 和持 SRS 的外国人）。

**可以观望**：本金不足 $1,000、追求短线交易、或已有一套低成本 DIY 指数基金方案且愿意自己维护再平衡的人。
