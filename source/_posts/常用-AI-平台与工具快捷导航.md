收藏常用 AI 平台与工具的快捷入口，方便日常查阅与跳转。

<!-- more -->

## 🤖 综合大模型与对话

* [ChatGPT](https://chatgpt.com) — 文本对话、提示词深度润色与重构。
* [Claude](https://claude.ai) — 代码编写、深度分析与高质量文本生成。
* [Grok](https://x.ai) — 具备 X 平台实时信息检索能力的强交互大模型。
* [OpenRouter](https://openrouter.ai/keys) — 聚合多种开源与闭源模型的 API 密钥与路由管理后台。
* [Qwen Studio](https://bailian.console.aliyun.com) — 阿里百炼大模型服务平台控制台，支持千问系列接入。
* [DeepSeek Platform](https://platform.deepseek.com) — DeepSeek API 管理与充值控制台。

## ☁️ 云计算与企业级 AI 平台

* [AWS Console](https://console.aws.amazon.com) — 亚马逊云科技管理控制台，直达 Amazon Bedrock 服务。
* [Google Cloud Console](https://console.cloud.google.com) — 谷歌云控制台，直达 Vertex AI 与企业级算力管理。
* [Azure Portal](https://portal.azure.com) — 微软云服务门户，用于配置与管理 Azure OpenAI 企业级服务。

## 🎬 视频与图像生成

* [Volcengine Console](https://console.volcengine.com) — 字节跳动火山引擎控制台，用于接入与测试 Seedream 等视频模型。
* [Google AI Studio](https://aistudio.google.com) — 谷歌 Gemini 多模态与 Veo 视频模型原生开发与测试控制台。
* [Higgsfield AI](https://higgsfield.ai) — 专注于角色驱动与高动态运动的移动端视频生成。
* [MiniMax Platform](https://platform.minimaxi.com) — MiniMax 开放平台控制台，管理各类大模型接口。
* [Runway Login](https://app.runwayml.com) — Runway 工作台，直达 Gen-3 视频生成项目管理。
* [Luma Dream Machine](https://lumalabs.ai/dream-machine) — 具备极强相机运动控制的高动态视频生成工具。
* [快手可灵 (Kling AI)](https://klingai.com) — 强大的文生视频与图生视频平台，物理模拟出色。
* [Midjourney](https://www.midjourney.com) — 顶尖的艺术与商业图像生成平台。
* [Black Forest Labs](https://blackforestlabs.ai) — SD 原班人马打造的 Flux 开源/商用图像大模型。

## 💻 AI 编程与 Agent 开发

* [Cursor Settings](https://cursor.com/settings) — 旗舰级 AI 代码编辑器个人面板，用于管理账户与模型调用配置。
* [OpenCode Auth](https://opencode.ai/auth) — 终端原生 AI 编码智能体（OpenCode）的鉴权与 API 密钥管理后台。
* [Trae](https://www.trae.sh) — 字节跳动推出的下一代 AI 编程 IDE。
* [Manus](https://manus.im) — 全能型通用 AI Agent 平台，全自动执行复杂任务。
* [YouMind](https://youmind.ai) — 专注于知识库、工作流与 Agent 构建的生产力工具。
* [Xiaomi MiMo](https://mimo.mi.com) — 小米大模型服务平台，赋能端侧编程助手与智能体生态。
* [Z.ai](https://z.ai) — 智谱 AI 海外旗舰，主打长周期自主编程与复杂工程任务。
* [BigModel.cn](https://bigmodel.cn/usercenter/apikeys) — 智谱 AI 开放平台控制台，直达 GLM 及 CogVideo API 密钥管理。

## 🎵 音乐与音频创作

* [Suno AI](https://suno.com) — 爆火的 AI 音乐生成平台，支持词曲唱一键创作。
* [Udio](https://www.udio.com) — 主打高保真音质与丰富音乐风格的高级音频工具。
