在新加坡办信用卡，Citi（花旗）是很多人绕不开的名字——返现卡、交通卡、里程卡都有，迎新礼也经常不差。但产品线一多，反而容易选错：**有人以为 SMRT 卡只能刷公交，其实网购 5% 才是大招；有人冲着 Cash Back 卡 8% 加油去，却忽略每月 $80 上限。**

这篇按 2026 年 Citi 官网与 [SingSaver](https://www.singsaver.com.sg/credit-card/citibank)、[MoneySmart](https://www.moneysmart.sg/credit-cards/citibank) 等第三方信息，把**最实用的几张卡**讲清楚：谁适合办、门槛多少、怎么配卡，以及文末如何用**邀请码**拿迎新礼。

## Citi 在新加坡有哪些「主力卡」？

可以把 Citi 零售信用卡粗分为三条线：

| 类型 | 代表卡 | 核心逻辑 |
| --- | --- | --- |
| **类别返现** | Cash Back、SMRT、M1 | 指定场景高返，其他场景很低 |
| **无脑返现** | Cash Back+ | 全品类固定比例，少算计 |
| **里程 / 积分** | PremierMiles、Rewards、Prestige | 换机票、酒店或 ThankYou 积分 |

对多数华人家庭和上班族，**真正值得认真比较的是：SMRT、Cash Back、Cash Back+，再加上一张里程卡（PremierMiles 或 Rewards）**。下面按「最常用 → 按需」展开。

## 1. Citi SMRT Card：网购 + 通勤的「5% 神卡」

官网：[Citi SMRT Card](https://www.citibank.com.sg/credit-cards/cashback/smrt-credit-card)

很多人以为它是「坐地铁的卡」，其实 2026 年最有价值的是：**达标后，杂货、网购、打车 / 网约车、SimplyGo 公交地铁、（限时）马来西亚令吉消费，都能拿到约 5% 的 SMRT$ 回馈。**

### 怎么赚

| 项目 | 说明 |
| --- | --- |
| 高返类别 | 超市 / 杂货、**线上消费**、出租车 / 网约车、**SimplyGo 点卡乘车**、MYR 消费（活动期） |
| 高返条件 | 该账单月零售消费满 **S$500**，否则上述类别也只有约 **0.3%** 基础回馈 |
| 其他消费 | 约 **0.3%** |
| 封顶方式 | **没有月封顶**，但会员年累计高返约 **SMRT$600**（约等于 **S$600** 现金价值） |
| 回馈单位 | SMRT$（通常可按 1:1 换现金返现或购物券） |
| 年费 | **S$196.20**（含 GST），新户常见 **首 2 年豁免** |
| 收入门槛 | 新加坡公民 / PR 年收入约 **S$30,000**；外籍约 **S$42,000** |

> **为什么说「无月封顶」很重要？**  
> 很多返现卡是「每月最高 $20–$80」。SMRT 卡把 **S$600 额度放在整年**，意味着你买一台几千刀的电器、付一笔大额网购，只要还在年额度内，**可以整笔吃 5%**。社区和第三方评测（如 Sethisfy、SingSaver）经常因此把它列为「大额线上付款」首选之一。

### 使用时容易踩的坑

1. **线上 ≠ 所有线上**  
   官网写明：线上高返**不含**手机钱包类、以及旅行相关交易。订机票 / 酒店若被归到 travel MCC，可能只有 0.3%。大额网购前，可先用小额测试或查 [Citi 不奖励商户 / MCC 列表](https://www.citibank.com.sg/rwdexcl/)。

2. **Grab 要用卡直接扣**  
   Grab 行程请绑卡直接支付；用 GrabPay 钱包再绑卡，通常**拿不到** 5% 加成。

3. **公交请用 SimplyGo 点卡**  
   新发的 SMRT 卡已不再内置 EZ-Link 储值；要拿高返，需用 **SimplyGo（信用卡直接拍闸机）**，不要依赖旧的 EZ-Link 自动充值逻辑。

4. **马来西亚令吉 5% 是活动**  
   目前官网活动期至 **2026 年 8 月 31 日**，适合常去新山 / JB 刷令吉的人；结束后以当时条款为准。

5. **兑换门槛**  
   SMRT$ 换现金通常有最低门槛（常见从 SMRT$10 起），需在 App / 短信兑换，不是「刷完自动到账」。

### 谁适合

- 每月网购 + 买菜 + 通勤能轻松过 **S$500**
- 偶尔有 **大额网购**（家电、3C、线上缴费类）
- 不在乎「每月都要算类别」，但愿意保证「满 500 才刷高返场景」

## 2. Citi Cash Back Card：加油、打车、吃饭、买菜的「日常高返」

官网：[Citi Cash Back Card](https://www1.citibank.com.sg/credit-cards/cashback/citi-cash-back-card)

如果你开车多、Grab 多、下馆子多，这张比 SMRT 更对口。

### 怎么赚（账单月满 S$800 时）

| 类别 | 返现 | 典型场景 |
| --- | --- | --- |
| 汽油 | **8%** | 加油站（MCC 5541 / 5542） |
| 私人通勤 | **8%** | Grab、Gojek、Comfort、Tada 等（MCC 4121） |
| 餐饮 | **6%** | 餐厅、快餐、部分外送归类 |
| 超市杂货 | **6%** | 超市 / 杂货店（MCC 5411） |
| 其他零售 | **0.2%** | 未进高返类别的消费 |
| 未满 S$800 | 全场约 **0.2%** | — |

- **月返现总上限：S$80**（所有类别合计）
- 返现**不过期**；累计满一定金额后按月自动入账（常见规则：满 S$50 起，按 S$10 整数倍入账）
- 年费 **S$196.20**，新户常见 **首年豁免**
- 配合 Caltex / Esso / Shell 等 Citi 加油优惠时，官网宣传综合加油节省可更高（含 8% 卡返 + 油站折扣，以当期活动为准）

### 和 SMRT 怎么选？

| | **Cash Back** | **SMRT** |
| --- | --- | --- |
| 达标门槛 | 月 **S$800** | 月 **S$500** |
| 最高场景 | 油 / 打车 8%，餐 / 菜 6% | 网购 / 菜 / 通勤 5% |
| 封顶 | **每月 S$80** | **每年约 S$600** |
| 大额网购 | 基本只有 0.2% | 可吃满 5%（在年额度内） |
| 更适合 | 油车 + 餐饮日常 | 网购 + 通勤 + 偶尔大额线上 |

一句话：**日常「吃喝开」看 Cash Back；「买买买 + 通勤」看 SMRT。** 预算够、场景分得开的人，**两张都办、分流刷**，往往比死磕一张更赚。

## 3. Citi Cash Back+ Card：不想动脑时的「全场 1.6%」

官网：[Citi Cash Back+ Card](https://www.citibank.com.sg/credit-cards/cashback/cashback-plus-card/)

适合：消费很散、不想记 MCC、也不想每月卡门槛。

### 亮点

- 合格零售消费约 **1.6% 返现**，**无最低消费、无月上限**
- 若你是 **Citi Plus** 客户，且是 **Citi Interest Booster Account** 主户，可再拿 **0.4% Bonus**（需当月达标，Bonus 有月封顶，常见约 S$8）
- 返现可在 Citi App / 短信兑换成现金返现，使用灵活
- 迎新常见形式：**前 2 个月指定消费给更高比例**（官网曾宣传最高约 8% 迎新返现，有消费与上限要求，需看当期 T&Cs）

### 要注意

- 1.6% 在新加坡「无脑卡」里属于中上水平，但**打不过** UOB Absolute、部分 1.7%–2% 竞品的峰值；它的优势是 Citi 生态 + 迎新 + 和 SMRT / Cash Back 搭配补位
- 账单代付、部分缴费类可能不给返现（以排除列表为准）
- 年费同样约 **S$196.20**，新户常见首年豁免

**适合谁：** 只想办一张 Citi、或者已经有 SMRT / Cash Back，需要一张「杂项兜底」卡。

## 4. Citi PremierMiles：里程不过期的旅行卡

官网入口见 [Citi 信用卡总览](https://www.citibank.com.sg/credit-cards)

| 项目 | 大致规则 |

| 本地消费 | 约 **1.2 Citi Miles / S$1** |
| 外币消费 | 约 **2.2 Citi Miles / S$1** |
| 里程过期 | **不过期**（Citi Miles 的核心卖点） |
| 外币手续费 | 约 **3.25%**（行业常见水平） |
| 年费 | 约 **S$196.20**，首年常豁免；续年有时附带续卡里程 / 贵宾室等福利 |
| 迎新（邀请活动） | 官网 Refer 页面常见 **最高约 30,000 bonus Citi Miles**（分档达标） |

### 适合谁

- 一年飞几次、愿意把外币消费集中在一张卡
- 不喜欢「积分年底清零」的焦虑
- 会用 **Citi PayAll** 等工具把部分大额账单换成里程（有手续费，需算净收益）

**不适合谁：** 几乎不出国、也不换机票——同样的消费用返现卡通常更「看得见」。

## 怎么配卡更划算？（实操建议）

### 方案 A：只办一张

| 画像 | 选 |
| --- | --- |
| 网购 + 地铁 / Grab 为主 | **SMRT** |
| 开车 + 下馆子为主 | **Cash Back** |
| 完全不想研究 | **Cash Back+** 或竞品无脑卡 |
| 主要出国刷 | **PremierMiles**（本地日常另配返现卡更好） |

### 方案 B：两张组合（更常见）

1. **SMRT + Cash Back**  
   - 网购买菜通勤 → SMRT  
   - 加油打车吃饭 → Cash Back  
   - 杂项看哪张当月还差门槛，用来「凑满」

2. **SMRT + PremierMiles**  
   - 本地日常 → SMRT  
   - 外币 / 订票 → PremierMiles  

3. **Cash Back+ 兜底 + 一张高返**  
   - 高返场景用 SMRT 或 Cash Back  
   - 其余一律 Cash Back+

### 方案 C：迎新优先

- 还不是 Citi 持卡人：优先看**官方邀请迎新** vs [SingSaver](https://www.singsaver.com.sg/credit-card/citibank) / [MoneySmart](https://www.moneysmart.sg/credit-cards/citibank) 礼包，**两边比完再申请**（一般**不能叠加**，详见 [SingSaver vs MoneySmart 指南](/2026/06/29/SingSaver-vs-MoneySmart：新加坡信用卡比价网站怎么选/)）。
- 已经是 Citi 持卡人：多数新卡迎新不给「新客礼」，但可留意升级 Prestige 等特殊通道。

## 常见问题

**Q：SMRT 和 Cash Back 可以一起办吗？**  
可以。两张卡场景重叠不多，很多人就是「一张刷网购，一张刷加油吃饭」。注意 Citi 对总授信、收入会综合审批。

**Q：返现是自动进账户吗？**  
- **Cash Back**：达标后按规则自动入账单（有最低累计与整数倍规则）  
- **SMRT**：先累积 SMRT$，再兑换成现金返现 / 券  
- **Cash Back+**：在 App 或短信兑换  

**Q：官网礼和大平台礼哪个好？**  
不固定。有时 SingSaver / MoneySmart 会「加码」现金或 Dyson 等实物；有时官方邀请码更简单、到账路径更短。**同一张卡申请一次**，建议申请前 5 分钟两边都打开比一遍。

**Q：外籍人士能办吗？**  
可以，收入门槛通常更高（约 S$42,000），并需护照、工作准证、住址证明等材料。
