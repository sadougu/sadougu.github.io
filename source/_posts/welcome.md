Welcome!

这是基于 [Hexo](https://github.com/hexojs/hexo) 构建的个人网站，部署在 GitHub Pages 上。

## Quick Start

### Create a new post

```bash
hexo new "My New Post"
```

### Run local server

```bash
npm run server
```

### Generate static files

```bash
npm run build
```

更多文档见 [Hexo Docs](https://hexo.io/docs/)。
